### 📂 Highlighted Technical Projects

#### 1. 🎵 PlaylistSync AI — Spotify to YouTube Music Conversion Agent
A production-grade playlist migration agent designed using a multi-process queue structure.
* **Tech Stack**: Next.js (React/Tailwind), Node.js (Express), BullMQ, Redis Cache, PostgreSQL (Prisma), Qdrant (Vector DB), Gemini API.
* **Key Feature**: Optimizes conversion costs and avoids YouTube search quotas by deploying a multi-layer decision pipeline (L1 Cache -> L2 SQL History -> L3 Qdrant Semantic RAG -> L4 Local scoring -> L5 Gemini AI validation).
* **Real-time Updates**: Implemented Socket.io WebSockets wired via Redis Pub/Sub channels to report live track conversion progress.

#### 2. 🛡️ DocuShield — Compliance Auditor RAG
A compliance-auditing contract analysis system.
* **Tech Stack**: Node.js, LangChain, Pinecone DB, Gemini API.
* **Key Feature**: Deployed a **Parent-Document Retrieval** pattern utilizing Gemini embeddings to eliminate LLM hallucination limits.
* **Optimization**: Automated Pinecone index configurations and bulk vector streaming, reducing auditing cycles from hours to seconds.

#### 3. 💬 Quick-Chat — Real-Time Messaging App
A full-stack, low-latency messaging platform.
* **Tech Stack**: Node.js, Express.js, MongoDB, Socket.io, Cloudinary.
* **Key Feature**: Supports sub-second message exchanges, JWT authorization session keys, Bcrypt secure passwords, and Cloudinary media sharing.

#### 4. 🤖 AI-Code-Reviewer — Automated Git Patcher
An automated repository auditing tool.
* **Tech Stack**: Node.js, Gemini API, GitHub REST API, Octokit.
* **Key Feature**: Utilizes an agentic tool-calling workflow to parse files locally, audit security flows, and push suggested code patches directly to GitHub repositories.
